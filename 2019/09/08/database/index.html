多行合并

当 log 输出多行时，kibana 上显示为多条，需要进行合并。日志以日期格式 yyyy-MM-dd 开头，配置 filebeat.yml 添加

pattern：正则表达式

negate：true 或 false；默认是false，匹配pattern的行合并到上一行；true，不匹配pattern的行合并到上一行

match：after 或 before，合并到上一行的末尾或开头
